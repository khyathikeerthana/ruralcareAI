# RuralCareAI Backend

FastAPI backend scaffold for role-based telemedicine workflows.

## Quick Start

1. Create a virtual environment.
2. Install dependencies from requirements.txt.
3. Copy .env.example to .env and fill your PostgreSQL credentials.
4. Run the API server.

## Run

Use uvicorn with app.main:app from the backend folder.

## Notes

- Never commit your .env file.
- This scaffold contains placeholder endpoints for incremental implementation.
- PostgreSQL schema is available at ../database/schema.sql.
